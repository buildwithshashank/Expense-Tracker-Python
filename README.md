# Expense Tracker 

A command-line expense tracker built in Python using OOP concepts.

## Features
- Add new expenses with amount and description
- View all expenses
- Calculate total expenses
- Delete specific expenses
- Simple menu-driven interface

## How to Run
python Tracker.py

## Concepts Used
- Object-Oriented Programming (Classes, Methods, self)
- Loops (while, for)
- User Input handling
- Type conversion (int())
